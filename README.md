## Zahtevi
- Node.js 18+
- npm

## Web aplikacija (Vite)
1) Instaliraj node pakete:
```
npm install
```
2) Pokreni dev server:
```
npm run dev
```

## Android aplikacija (Expo Go)
1) Instaliraj node pakete:
```
cd mobile
npm install
```
2) Pokreni aplikaciju:
```
npm run start
```
